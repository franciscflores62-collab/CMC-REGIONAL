Carpeta para recursos estáticos del sitio (logo, fondo, íconos propios).

Coloca aquí, con estos nombres exactos, para que index.html los detecte automáticamente:
  - logo.png         -> Logo CMC / Gildan (cuadrado, fondo transparente recomendado)
  - background.jpg   -> Imagen de fondo del panel principal

Si estos archivos no existen, el sitio usa automáticamente un ícono y un fondo de respaldo, así que nunca se rompe la vista para quien lo visite en GitHub Pages.
